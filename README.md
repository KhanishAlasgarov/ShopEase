# ShopeEase (E-commerce app)

### This is a clothing e-commerce app built for Android, allowing users to browse, search, and purchase clothing items. The app includes features like user authentication, shopping cart management, and order tracking. The app is developed using Kotlin, Firebase for backend services, and XML for UI.

## Project Features
 - MVVM
 - Material Design 3
 - Coroutines & Flow
 - Hilt for Dependency Injection
 - Firebase
    - Firebase Auth
    - Firestore Database
    - Storage
 - Android JetPack
    - LiveData
    - Lifecycle
    - ViewModel
    - View Binding
    - DataBinding
 - Navigation Components
 - ROOM Database
 - Glide 
 - Lottie 
 - FancyToast
 - ViewPager2
